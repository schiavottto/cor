HTML e CSS Explorados

Tags HTML

<p>

A tag <p> é usada para definir um parágrafo de texto em HTML. Ela cria uma quebra de linha antes e depois do conteúdo do parágrafo.

<img>

A tag <img> é usada para inserir uma imagem em uma página HTML. Ela requer um atributo src para especificar o caminho da imagem e pode incluir atributos opcionais como alt para texto alternativo e width e height para especificar as dimensões da imagem.

<h1>

As tags de cabeçalho <h1> a <h6> são usadas para definir cabeçalhos de diferentes níveis de importância em um documento HTML. <h1> é o mais importante e <h6> o menos importante.

Propriedades CSS

Tamanho

A propriedade CSS font-size define o tamanho do texto dentro de um elemento HTML. Pode ser especificada em unidades como pixels (px), ems (em), porcentagem (%), entre outras.

Centralização

Para centralizar elementos em CSS, você pode usar a propriedade text-align: center; para textos e margin: 0 auto; para elementos de bloco como <div>.

Cor

A cor do texto e dos elementos pode ser definida usando a propriedade color. Pode ser especificada utilizando nomes de cores (como red, blue) ou códigos hexadecimais (#RRGGBB).
